## Implementation Notes

### Docker Compose Setup

**File:** `docker-compose.yml`

```yaml
version: '3.8'

services:
  executor:
    build:
      context: .
      dockerfile: Dockerfile
    ports:
      - "7999:7999"
    environment:
      - PYTHONUNBUFFERED=1
      - EXECUTOR_API_KEY=${EXECUTOR_API_KEY}
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:7999/health"]
      interval: 10s
      timeout: 5s
      retries: 3

volumes:
  executor_tmp:
```

### Python Dockerfile

**File:** `docker/python/Dockerfile`

```dockerfile
FROM python:3.11-slim

WORKDIR /workspace

RUN apt-get update && apt-get install -y --no-install-recommends \
    build-essential && rm -rf /var/lib/apt/lists/*

RUN pip install --no-cache-dir pytest==7.4.3 pytest-timeout==2.2.0

RUN useradd -m -u 1000 executor
USER executor

CMD ["/bin/bash"]
```

### JavaScript Dockerfile

**File:** `docker/javascript/Dockerfile`

```dockerfile
FROM node:20-slim

WORKDIR /workspace

RUN npm install -g --silent jest@29.7.0 mocha@10.2.0

RUN useradd -m -u 1000 executor
USER executor

CMD ["/bin/bash"]
```
