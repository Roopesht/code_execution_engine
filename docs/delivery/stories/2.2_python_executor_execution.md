## Implementation Notes

### Docker Execution Command

```bash
docker run \
  --rm \
  -v /tmp/executor_<uuid>:/workspace \
  --memory=512m \
  --cpus=0.5 \
  --timeout=5 \
  python:3.11-slim \
  pytest /workspace/test_solution.py -v --tb=short
```

### Resource Limits

- Memory: 512 MB
- CPU: 0.5 cores
- Timeout: 5 seconds (configurable)
