[← Back to Scope Matrix](../01_scope-matrix.md)

---

**Phase:** 3 - API Integration  
**Priority:** High  
**Dependencies:** Story 2.1-2.6

---

## Description

Implement the main POST `/execute` endpoint that orchestrates the entire execution workflow.

---

## Tasks

- [ ] Create POST `/execute` endpoint
- [ ] Route requests to appropriate executor (Python/JavaScript)
- [ ] Call executor methods: prepare → execute → collect_results → cleanup
- [ ] Handle workflow errors
- [ ] Ensure workspace cleanup on error
- [ ] Return formatted response

---

## Acceptance Criteria

- [ ] Endpoint accepts POST requests
- [ ] Routes to correct executor based on language
- [ ] Full workflow executes successfully
- [ ] Workspace created and cleaned up
- [ ] Response returned in < 3 seconds
- [ ] Errors don't leave orphaned containers/files
- [ ] Concurrent requests handled correctly

---

## Success Criteria

✅ Endpoint works correctly  
✅ Routing works  
✅ Cleanup happens
