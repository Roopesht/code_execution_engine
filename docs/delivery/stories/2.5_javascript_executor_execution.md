## Description

Implement JavaScript code execution within Docker containers using jest.

---

## Tasks

- [ ] Implement `execute()` method for JavaScript
- [ ] Start Docker container with Node.js environment
- [ ] Run jest with proper configuration
- [ ] Capture output (stdout/stderr)
- [ ] Record execution metrics
- [ ] Handle timeouts
- [ ] Clean up container

---

## Acceptance Criteria

- [ ] Execution runs in isolated container
- [ ] Jest executes successfully
- [ ] Output captured completely
- [ ] Execution time < 2 seconds
- [ ] Memory usage recorded
- [ ] Container destroyed after execution
- [ ] Timeout handled (5s default)
- [ ] No resource leaks

---

## Success Criteria

✅ Code executes in container  
✅ Output captured  
✅ Metrics recorded
